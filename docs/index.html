ZW3021 設定

指紋センサー: USBまたはBluetooth経由で指ごとの出力文字列と登録を管理します。

Chrome または Edge 89以上、かつsecure context(HTTPS・localhost・ ローカルファイル)が必要です。

USBで接続: ZW3021センサーが付いているキーボードの 半分をUSB接続してからクリックし、ブラウザのポート選択ダイアログで そのシリアルポートを選んでください(Web Serial API)。

Bluetoothで接続: センサー側キーボードの電源が 入っていれば配線不要です。クリックしてデバイス選択ダイアログで このキーボードを選んでください(Web Bluetooth API)。初回接続時は OSのペアリングが必要になることがあります。

指スロット

「文字列設定済み/未設定」は出力文字列がNVSに保存されて いるかどうか、「指紋登録済み/指紋未登録」はセンサー 本体に指紋テンプレートが保存されているかどうかを表す、 互いに独立した情報です(出力文字列だけ設定して指紋登録前、ということ もあり得ます)。指紋側は「一覧を更新」を押した時にセンサーへ問い合わせて 取得するため、直後は「確認中」と表示されることがあります。

この画面から保存済みの文字列そのものを読み出すことは できません(書き込み専用)。 このシリアルコンソールには認証がなく、USB物理接続さえできれば 誰でもコマンドを送れるため、値をそのまま返す設計にすると指紋 認証を経由せずに全部読み出せてしまいます。それを防ぐための 仕様です。保存は常に新しい値での上書きのみになります。

指 1

「指を登録」はセンサーへの指紋登録(PS_AutoEnroll)、 「文字列を削除」は保存済みの出力文字列とEnter設定を消すだけです。 指紋テンプレート自体を消すには実機の &zw3021_delete キーマップキーを使ってください(このWeb UIからは行えません)。

出力文字列について

センサー本体のフラッシュ(NVS)にのみ保存されます — このシリアル 接続以外でデバイスの外に出ることはなく、このリポジトリに 書き込まれることもありません。
